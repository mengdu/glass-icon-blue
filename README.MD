## glass-icon-blue
   一个win7系统的蓝色玻璃图标集合。
   来源于 [win7之家](http://www.win7china.com/)
## 使用方法

首先请把imageres.dll替换C:\Windows\System32中的imageres.dll

然后执行以管理员方式执行“清楚win7图标缓存.bat”，如果没效果请重启电脑。
